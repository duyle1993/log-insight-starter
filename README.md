# Log Insight Starter

A beginner cybersecurity project built with Python that analyzes authentication logs and identifies failed login attempts from suspicious IP addresses.

## Features
- Reads a log file
- Detects failed password attempts
- Extracts suspicious IP addresses
- Counts repeated failed logins
- Assigns a simple risk level

## Technologies Used
- Python

## How to Run
1. Place the log file in the project folder
2. Run:
   python log_analyzer.py

## Example Output
- Total failed login attempts
- Unique suspicious IPs
- Risk level for each IP

## Purpose
This project demonstrates basic log analysis and introduces security event detection for beginner SOC-style work.
